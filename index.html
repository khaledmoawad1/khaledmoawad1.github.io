Voice-to-Voice Chatbot

Developed a voice-interactive chatbot with a multi-stage pipeline: Speech-to-Text, RAG system powered by an LLM, and Text-to-Speech. EKYC National ID OCR System

Developed a custom OCR pipeline for Egyptian national ID cards using specialized text detection and three dedicated recognition models. Achieved high accuracy outperforming Qwen and Azure OCR. Integrated liveness detection and fraud detection mechanisms. Deployed using FastAPI.

Sales Data Insights Dashboard

AI Trainer - Exercise Form Analysis

Multiclass ROP Classification
